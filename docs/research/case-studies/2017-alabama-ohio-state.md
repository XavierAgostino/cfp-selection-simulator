# Case Study: 2017 Alabama and Ohio State

## Context

Alabama made the 4-team playoff despite not winning the SEC championship. Ohio State won the Big Ten but missed the field. The debate centered on resume vs conference championship and eye-test predictive strength.

## Simulator relevance

Use historical validation for 2017 to compare model rankings vs published CFP rankings. The composite pipeline separates **resume** (Colley + win%) from **predictive** (Massey + Elo) components — useful for post-hoc analysis of this decision.

## Questions to explore

- Component view: was Ohio State higher on resume and Alabama on predictive (or vice versa)?
