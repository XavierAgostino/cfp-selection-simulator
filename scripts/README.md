# Scripts

Utility scripts for the CFP Selection Simulator.

---

## `demo.sh`

One-shot demo for new users:

```bash
./scripts/demo.sh
```

Installs the package, runs `cfp-sim doctor`, executes sample pipeline, generates bracket HTML, and lists outputs.
