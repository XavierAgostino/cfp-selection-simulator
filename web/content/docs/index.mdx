Selection Room is a transparent CFP selection simulator. It projects fields, explains bubble logic, and surfaces bracket clarity from reproducible ranking inputs.

## What this page covers

- The purpose of Selection Room
- How product pages and docs relate
- Where to go next

## Who this is for

Analysts, fans, and developers who want to understand **how** a projected field is built, not just what the top four look like. The web app is the primary surface; these docs explain setup, methodology, and contracts.

## Product vs docs

| Surface | Purpose |
|---------|---------|
| [Dashboard](/) | Active run overview |
| [Methodology](/methodology) | In-app metric reference |
| [Validation](/validation) | Historical backtest dashboard |
| [Scenario Lab](/scenario-lab) | Weight sensitivity experiments |
