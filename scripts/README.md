# Scripts

Utility scripts for the Selection Room.

---

## `demo.sh`

One-shot demo for new users:

```bash
./scripts/demo.sh
```

Installs the package, runs `sroom doctor`, executes sample pipeline, generates bracket HTML, and lists outputs.
